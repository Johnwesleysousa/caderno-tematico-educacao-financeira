# Caderno Temático – Educação Financeira Introdutória

## Descrição do Projeto
Este projeto consiste na criação de um caderno temático de educação financeira introdutória, elaborado com o apoio da ferramenta NotebookLM para agregar e analisar fontes abertas. Contém objetivos de estudo, perguntas estratégicas, resumos, glossário, e prompts para estudo ativo. O trabalho valoriza a aprendizagem ativa e o uso da IA para organizar conhecimento financeiro.

## Estrutura do Caderno
1. **Objetivos**
   - Geral e específicos sobre educação financeira pessoal.
2. **Perguntas Estratégicas**
   - Seis perguntas para aprofundar conceitos e análises críticas.
3. **Resumo e Mini Guia**
   - Conceitos principais, plano mensal de organização financeira.
4. **Glossário**
   - Principais termos financeiros explicados de forma simples.
5. **Prompts para Revisão**
   - Perguntas para estudo ativo e revisão futura.
6. **Reflexão**
   - Aprendizado ativo usando NotebookLM e análise crítica das fontes.
